Readme file for TestVagrant coding assessment Programs Using Rest Assured & Java
----------------------------------------------------------------------------------------------------
java 1.8.0_321
RestAssured 4.3

Follow the below steps to execute the Project -

1. Clone Project in your local machine.

2. Provide TestData in the excel file, path -  src/main/java/testData/playerDetails.xlsx

3. Data format : Team - team name(RCB), playerData - Team players data in json format.

4. In config.properties file configure the max limit for foreign playes in "foreignPlayerLimit"
and number of wicket keepers required in "wicketKeeper" path - src/main/java/config/config.properties

5. Run TestNG.xml

6. Reports will be generated inside "test-output" directory in html format

Thank you
